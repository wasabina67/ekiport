# Marker

## Overview
Mark stations and airports with different icons.

## Requirements
1. [ ] Station marker
2. [ ] Airport marker

### Station marker
- Use `#4ecdc4` colored icon
- emoji: 🚉
- Click the marker to display name and detail (Pop-up)

### Airport marker
- Use `#f4845f` colored icon
- emoji: ✈️
- Click the marker to display name and detail (Pop-up)
